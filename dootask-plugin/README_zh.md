# AI 智能体插件

## 插件简介

AI 智能体插件为 DooTask 提供企业级 AI 智能助手能力，集成多款主流大语言模型，包括：

- **ChatGPT**（OpenAI）
- **Claude**（Anthropic）
- **DeepSeek**
- **Gemini**（Google）
- **Grok**（xAI）
- **Ollama**（支持本地部署模型）
- **Azure OpenAI**（企业级部署）
- **通义千问** 等国产模型

安装后，用户可在 DooTask 中创建多个专业的 AI 智能体，配置不同的角色、工具和知识库，用于客服助手、项目管理、知识检索、团队协作等多种场景，显著提升团队效率和智能化水平。

## 功能亮点

- **多智能体管理**：支持创建和管理多个不同角色的 AI 智能体，每个智能体可独立配置
- **实时对话体验**：支持流式回复、群聊对话、图片识别，体验流畅自然
- **知识库系统**：支持 PDF、Word、Excel、Markdown 等多种格式，基于向量检索的智能问答
- **MCP 工具集成**：支持内部工具（DooTask 功能）和外部工具（天气、搜索等），自动关联dootask MCP 服务
- **思考过程显示**：支持思考型模型（如 DeepSeek），显示 AI 的思考过程，提升透明度
- **企业级特性**：完整的权限管理、审计日志、Token 统计、数据安全保障
- **原生集成**：深度集成 DooTask 机器人系统，使用体验自然流畅
- **零配置上手**：默认参数已优化，支持自动关联配置，快速开始使用

## 安装说明

1. 在 DooTask 应用商店中找到并安装本插件
2. 安装完成后插件将自动启用，首次使用需要配置 AI 模型的 API 密钥
3. 插件默认配置已适配大多数使用场景，如有需要可在插件设置中进行自定义调整
4. 在智能体管理页面创建您的第一个 AI 智能体，开始享受智能化的团队协作体验

⚠️ **注意事项**：
- 插件需要 DooTask >= 1.1.66 版本
- 需要配置至少一个 AI 模型的 API 密钥才能正常使用
