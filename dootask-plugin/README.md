# AI Agent Plugin

## Overview

The AI Agent Plugin provides enterprise-grade AI assistant capabilities for DooTask, integrating multiple mainstream large language models, including:

- **ChatGPT** (OpenAI)
- **Claude** (Anthropic)
- **DeepSeek**
- **Gemini** (Google)
- **Grok** (xAI)
- **Ollama** (supports locally deployed models)
- **Azure OpenAI** (enterprise deployment)
- **Qwen** and other domestic models

After installation, users can create multiple professional AI agents in DooTask, configure different roles, tools, and knowledge bases for various scenarios such as customer service assistants, project management, knowledge retrieval, and team collaboration, significantly improving team efficiency and intelligence levels.

## Features

- **Multi-Agent Management**: Create and manage multiple AI agents with different roles, each independently configurable
- **Real-time Chat Experience**: Streaming responses, group chat, and image recognition for a smooth and natural experience
- **Knowledge Base System**: Support for PDF, Word, Excel, Markdown, and more, with intelligent Q&A based on vector retrieval
- **MCP Tool Integration**: Support for internal tools (DooTask features) and external tools (weather, search, etc.), automatically connected to DooTask MCP services
- **Thinking Process Display**: Support for thinking models (e.g., DeepSeek), showing AI thinking process for improved transparency
- **Enterprise Features**: Complete permission management, audit logs, token statistics, and data security
- **Native Integration**: Deeply integrated with DooTask bot system for a natural and smooth user experience
- **Zero-Config Quick Start**: Optimized default parameters with auto-configuration support for quick setup

## Installation

1. Find and install this plugin from the DooTask App Store
2. The plugin will be automatically enabled after installation; first-time use requires configuring AI model API keys
3. Default plugin settings are adapted for most use cases; customize in plugin settings if needed
4. Create your first AI agent in the Agent Management page to start enjoying intelligent team collaboration

⚠️ **Note**:
- Plugin requires DooTask >= 1.1.66
- At least one AI model API key must be configured for normal use
